Exercício 03: Parágrafo Colorido

Primeiro Parágrafo

Aqui iremos criar um parágrafo de estilo padrão com a cor de fundo cinza claro. Será usado o código Hexadecimal, usado em cores consiste em seis letras ou números precedidios de '#', e seus números significam:
os dois primeiros elementos representam a intensidade de vermelho;
o Terceiro e quatro elementos representam a intensidade de verde;
já os dois ultimos números a intensidade do azul. Assim, a sintaxe para definir a cor de fundo cinza claro será: backra: #F5F5F5.

Segundo Parágrado

Neste segundo parágrafo iremos usar o mesmo tipo de código Hexadecimal, só que a cor destacada no parágrafo será com o fundo dolorido, na cor rosa.
Uma curiosidade sobre as cores hexadecimal é que são cores exibidas em monitores e telas são definidas a partir do código hexadecimal, que é composto do sinal de sustenido (#) mais seis dígitos, como já citados no parágrafo primeiro acima.
